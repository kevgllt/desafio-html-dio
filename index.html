HTML

HTML (HyperText Markup Language) é uma linguagem fundamental para a construção de páginas da web.
É responsável por estruturar o conteúdo e definir como ele será exibido em um navegador.
Ao utilizar o HTML, é possível criar documentos que contêm textos, imagens, vídeos e outros elementos interativos.

Uma das principais características do HTML é a sua capacidade de organizar e hierarquizar o conteúdo. Isso é feito através da utilização de elementos e suas respectivas tags.
Cada tag tem uma função específica, como indicar um título, um parágrafo ou uma imagem.

Além disso, o HTML permite a criação de links, formulários, tabelas e outras estruturas que facilitam a navegação e a interação do usuário com o conteúdo.
Com a adição de atributos, é possível fornecer informações adicionais aos elementos, como o endereço de uma imagem ou o destino de um link.

Para estilizar o conteúdo HTML e torná-lo visualmente atraente, é comum utilizar o CSS (Cascading Style Sheets), que permite definir cores, fontes, tamanhos e outros estilos. Combinado com o HTML, o CSS possibilita a criação de páginas web com aparência profissional.

O HTML também pode ser complementado com o uso de JavaScript, uma linguagem de programação que adiciona interatividade e funcionalidades dinâmicas às páginas. Com o JavaScript, é possível criar animações, validar formulários, realizar requisições a servidores e muito mais.

No final, o HTML desempenha um papel crucial na construção de páginas da web, fornecendo a estrutura básica para a apresentação do conteúdo e a interação com os usuários.
É uma linguagem indispensável para aqueles que desejam criar e desenvolver sites e aplicações web.

Tags

^ subir para Indices

A estrutura de um documento HTML é formada da sequinte maneira;

<html>
<head></head>
<body></body>
</html>

Abaixo sera listado algumas tags e no fim desse indice um exemplo de como utilizalas.

Tag h1 comumente usada para criar titulos.

<h1></h1>

Alterando a numeração da tag pode ir de h1 a h6, conforme o numero aumenta o tamanho da fonte do titulo diminui.
Lembre-se de respeitar a sitaxe, forma correta da criação da tag, como o exemplo abaixo:

<h1></h1>
<h2></h2>
<h3></h3>
...

Forma incorreta, maneira que não ira funcionar:

<h1></h5>
<h3></h4>

<h1></h5>
<h3></h4>

Tag p para criar um paragrafo.

<p></p>

Tag a utilizada para criar links.
(Sera necessário utilizar alguns atributos para o funcionamento dessa tag, no exemplo de utilização a mesma sera apresentada)

<a></a>

Tag img para adicionar imagens.
(Sera necessário utilizar alguns atributos para o funcionamento dessa tag, no exemplo de utilização a mesma sera apresentada)

<img>

Estrurua e utilização das tags na prática e uso de alguns atributos para utilização das tags "a" (links para direcionar para outra web page) e "img" (adicionar imagens).


            
                <html>
                    <head>
                    <title>nome da aba</title>
                    </head>
                    <body>
                        <h1>Meu titulo</h1>
                        <h2>Meu subtítulo</h2>
                        <p>Meu paragrafo</p>
                        <a href="https://www.google.com.br/">Google</a>
                        <img src="https://www.google.com.br/images/branding/googlelogo/2x/googlelogo_color_272x92dp.png" alt="Logo do Google"/>
                    </body>
                </html>
